# State Overview
### Understanding state as the structural anchor of the REID Cognitive Architecture

State is one of the four core invariants of REID.  
It represents the current configuration, condition, or role of a system,
person, or domain. State is the anchor of all reasoning — the starting
point from which forces act, flows move, and horizons are defined.

This document explains what state is, how it behaves, and why it matters.

---

## 1. What State Is
State is **what exists right now**.

It includes:
- Current conditions  
- Current roles  
- Current configuration  
- Current context  
- Current capabilities  
- Current limitations  

State is the snapshot of the system at this moment.

---

## 2. Why State Matters
State determines:
- What forces can act  
- What flows are available  
- What horizons are realistic  
- What transitions are possible  

Without understanding state, reasoning becomes guesswork.

---

## 3. Types of State

### **A. Physical State**
Tangible conditions:
- System health  
- Environment  
- Physical resources  

### **B. Cognitive State**
Mental or emotional conditions:
- Stress  
- Clarity  
- Fatigue  
- Focus  

### **C. Structural State**
System configuration:
- Architecture  
- Rules  
- Policies  
- Organizational design  

### **D. Relational State**
Position relative to others:
- Trust  
- Alignment  
- Conflict  
- Influence  

### **E. Temporal State**
Where the system is in time:
- Early  
- Mid‑trajectory  
- Late  
- Transitioning  

---

## 4. State Stability
State can be:
- Stable  
- Unstable  
- Volatile  
- Transitional  
- Locked  
- Drifting  

Stability determines how forces and flows behave.

---

## 5. State Transitions
State changes when:
- Forces exceed resistance  
- Flows increase or decrease  
- Horizons shift  
- New information arrives  
- External conditions change  

State transitions are structural, not emotional.

---

## 6. State and the Other Invariants

### **Forces**
Forces act on state.  
State determines how forces are absorbed or resisted.

### **Horizon**
The horizon defines what the state should become.

### **Flows**
Flows enable state change.  
Flow scarcity slows transitions.

State is the anchor of the system.

---

## 7. Human State
Humans have states such as:
- Overwhelmed  
- Focused  
- Exhausted  
- Motivated  
- Conflicted  
- Stable  
- Drifting  

Human state determines:
- What horizons are possible  
- How forces are interpreted  
- How flows are used  

---

## 8. Multi‑Human State Systems
In multi‑human environments:
- Each person has their own state  
- States influence each other  
- Misaligned states create friction  
- Aligned states create stability  

REID models these interactions structurally.
